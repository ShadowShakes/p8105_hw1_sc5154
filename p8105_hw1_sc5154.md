Let’s create a data frame comprised of the required variable types.

``` r
data_vt = tibble(
  x_1 = rnorm(10, mean = 0, sd = 1),
  x_2 = x_1 > 0,
  x_3 = c('a','b','c','d','e','f','g','h','i','j'),
  x_4 = factor(c('a','a','a','a','b','b','b','b','c','c'),
        levels = c('a','b','c'))
)
```
